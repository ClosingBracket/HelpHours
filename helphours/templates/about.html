Why all the Ducks?

The rubber duck theme is inspired by "Rubber Duck Debugging," a technique for debugging code where you take the time to explain what your code does, line by line, to a rubber duck. Eventually, by explaining it out load, you'll notice a discrepency between what you expect the code does and what it is actually doing. Read more about it here!
